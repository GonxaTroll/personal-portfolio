# Personal Portfolio

Source code for my personal portfolio, hosted on GitHub Pages.

## What's in here

A single-page app built with React + TypeScript that covers:

- **About** — short intro, core stack, and links
- **Experience** — timeline of past roles with expandable detail cards
- **Projects** — selected personal and professional projects
- **Contributions** — research publications, open-source tooling, and Kaggle competitions
- **Certifications** — credential carousel

## Stack

- React 19 + TypeScript
- Tailwind CSS v4
- Radix UI (Dialog)
- Framer Motion
- Lucide icons
- Vite

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`, which is what gets deployed to GitHub Pages.
